CSS Grid

Grid is a two-dimensional layout: define rows and columns, and optionally areas. The parent is the grid container; direct children are grid items.

Rows and columns – grid-template-rows / grid-template-columns

Define the track sizes. 1fr = one fraction of available space. Use repeat(n, 1fr) for equal columns.

CSS 
.grid-basic {
  display: grid;
  grid-template-rows: auto auto;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 0.5rem;
  padding: 0.75rem;
  background: #d1fae5;
  border-radius: 8px;
}

.grid-cell {
  background: #059669;
  color: white;
  padding: 0.75rem;
  text-align: center;
  border-radius: 6px;
}

Named areas – grid-template-areas

Give names to regions in the grid, then assign items to those areas with grid-area. Great for page layouts (header, sidebar, main, footer).

Live demo
Header
Sidebar
Main
Footer
CSS 
.grid-areas {
  display: grid;
  grid-template-rows: auto 1fr auto;
  grid-template-columns: 120px 1fr;
  grid-template-areas:
    "header header"
    "side   main"
    "footer footer";
  gap: 0.5rem;
  min-height: 180px;
}

.area-header { grid-area: header; }
.area-side   { grid-area: side; }
.area-main   { grid-area: main; }
.area-footer { grid-area: footer; }

fr units and repeat()

1fr = one fraction of free space. repeat(3, 1fr) is shorthand for 1fr 1fr 1fr.
