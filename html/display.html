CSS Display

The display property controls how an element is laid out: block (stacked, full width), inline (flows with text), flex, and grid (layout containers).

Block – full width, stacked vertically

Block elements start on a new line and take the full width of their parent. Default for div, p, section, etc.

CSS 
.block-item {
  display: block;
  background: #cbd5e1;
  padding: 0.5rem 1rem;
  margin: 0.25rem 0;
  border-radius: 4px;
}

Inline – flows with text

Inline elements stay in the line of text; width and height don’t apply the same way. Default for span, a, strong.

Live demo

Here is inline 1 and inline 2 and inline 3 in a sentence.

CSS 
.inline-item {
  display: inline;
  background: #a5b4fc;
  padding: 0.15rem 0.5rem;
  border-radius: 4px;
  margin: 0 0.15rem;
}

Flex – one-dimensional layout

The parent becomes a flex container; children become flex items arranged in a row or column. See the Flexbox page for more.

2
3
CSS 
.flex-demo {
  display: flex;
  gap: 0.5rem;
  padding: 0.5rem;
  background: #e0e7ff;
  border-radius: 6px;
}

.flex-item {
  flex: 1;
  background: #6366f1;
  color: white;
  padding: 0.75rem;
  text-align: center;
  border-radius: 4px;
}
