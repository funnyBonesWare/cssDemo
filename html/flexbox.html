CSS Flexbox

The parent is the flex container (display: flex); direct children are flex items. Use justify-content for the main axis and align-items for the cross axis.

Container: display: flex

The flex container lays out its children in a row by default. Items can grow, shrink, and align along the main axis (row) and cross axis (column).

CSS 
.flex-container {
  display: flex;
  gap: 0.5rem;
  padding: 0.75rem;
  background: #e0e7ff;
  border-radius: 8px;
}

.flex-item {
  background: #6366f1;
  color: white;
  padding: 0.75rem 1rem;
  border-radius: 6px;
  min-width: 48px;
  text-align: center;
}

justify-content – main axis alignment

Controls how items are distributed along the main axis (horizontal when flex-direction is row). Try the buttons below.

Live demo
1
2
3
CSS 
.flex-container {
  display: flex;
  justify-content: flex-start; /* or center | flex-end
    | space-between | space-around | space-evenly */
  gap: 0.5rem;
  padding: 0.75rem;
  background: #e0e7ff;
  border-radius: 8px;
}

align-items – cross axis alignment

Controls how items align along the cross axis (vertical when in a row). Options: stretch (default), flex-start, center, flex-end, baseline.

Live demo (align-items: center)

CSS 
.flex-container {
  display: flex;
  align-items: center; /* stretch | flex-start
    | center | flex-end | baseline */
  min-height: 100px;
}

.flex-item-tall {
  padding: 1.5rem 1rem; /* one item taller */
