CSS Box Model

Every element is a box. From inside out: content → padding → border → margin. Padding is inside the element (background applies); margin is outside (space between elements).

Visual breakdown

The outer dashed border shows the element’s margin area; the yellow band is padding; the blue rectangle is the content.

Content

Content Padding Border Margin

CSS 
.box-demo {
  margin: 2rem 0;        /* outside */
  border: 4px dashed #f59e0b;
  padding: 24px;          /* inside */
  background: #fef3c7;   /* padding area */
}

.box-content {
  background: #1e3a8a;   /* content area */
  color: white;
  padding: 1rem;
}

Properties

  • width / height – by default, the size of the content area only.
  • padding – space between content and border; background and border-radius apply here.
  • border – line around the padding.
  • margin – space outside the border; not part of the element’s background.
box-sizing: border-box makes width and height include padding and border, so the total size is easier to reason about. Many projects set it globally with * { box-sizing: border-box; }.
CSS 
* {
  box-sizing: border-box;
}

.box-a {
  width: 200px;
  padding: 24px;   /* more padding = bigger content area */
  margin: 10px;
  border: 2px solid #2563eb;
}

Padding vs margin

Box A has more padding (inner space); Box B has more margin (outer space).
