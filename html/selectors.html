CSS Selectors

Selectors determine which elements get which styles. Types: #id, .class, element, [attribute], and combinators: , (group),   (descendant), > (child), + (adjacent sibling).

Element selector – element

Targets every element of that tag name (e.g. all p, all span).

Live demo

This paragraph is styled by the p selector.

This span is styled by the span selector.
CSS 
p {
  margin: 0.5rem 0;
}

span {
  display: inline-block;
  margin: 0.5rem 0;
}

Class selector – .class

Targets elements whose class attribute contains the given name. Multiple elements can share a class.

Live demo

Paragraph with class highlight.

Paragraph without the class.

Another highlight paragraph.

CSS 
.highlight {
  background: #fef3c7;
  padding: 0.25rem 0.5rem;
  border-left: 3px solid #f59e0b;
}

ID selector – #id

Targets the single element with that id. IDs must be unique per page; prefer classes for styling.

Live demo

This paragraph has id="main-intro".

CSS 
#main-intro {
  background: #dbeafe;
  padding: 0.5rem 1rem;
  border-radius: 6px;
  font-weight: 500;
}

Attribute selector – [attribute]

Targets elements that have the attribute (and optionally a given value). Useful for links, inputs, etc.

Live demo Link with href
Link without href
CSS 
a[href] {
  color: #2563eb;
  text-decoration: underline;
}

input[type="text"] {
  border: 2px solid #2563eb;
  padding: 0.25rem 0.5rem;
  border-radius: 4px;
}

input[type="submit"] {
  background: #2563eb;
  color: white;
  border: none;
  padding: 0.35rem 1rem;
  border-radius: 4px;
  cursor: pointer;
}

Descendant combinator – parent child (space)

Matches any child that lives inside parent, at any nesting level.

Live demo

Direct child paragraph

Nested paragraph (also matched)

CSS 
.parent-desc p {
  background: #e0e7ff;
  padding: 0.25rem 0.5rem;
  margin: 0.25rem 0;
  border-radius: 4px;
}

Child combinator – parent > child

Matches only direct children, not deeper descendants.

Live demo

Direct child only (matched)

Nested (not matched)

CSS 
.parent-child > p {
  background: #c7d2fe;
  padding: 0.25rem 0.5rem;
  border-radius: 4px;
}

Adjacent sibling – a + b

Matches b only if it immediately follows a (same parent).

Live demo

Heading

First paragraph after h4 (matched)

Second paragraph (not matched)

  background: #bbf7d0;
  padding: 0.25rem 0.5rem;
  border-radius: 4px;
}
